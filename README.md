1. Install the mod in Mindustry
2. Click the **"ModInfo+"** button in the main menu
3. Browse through different categories using the category buttons
4. Navigate pages with **"< Prev"** and **"Next >"** buttons
5. Click **"Refresh"** to update statistics (60-second cooldown)

## Requirements

- Mindustry (latest version recommended)
- Internet connection for fetching GitHub stats

## Installation

1. Download the latest `.zip` file from [Releases](../../releases)
2. Place it in your Mindustry `mods` folder
3. Restart Mindustry
4. The "ModInfo+" button will appear in the main menu

## How It Works

ModInfo+ uses the GitHub API to fetch release information and calculates total downloads across all releases for each tracked mod. All data is fetched in real-time when you open the stats viewer.
